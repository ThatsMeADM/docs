---
api: "GET /api/v1.0/balance"
title: "Saldo"
---

Retorna o **saldo atual de créditos** da conta emissora autenticada.

<Request method="GET" url="/api/v1.0/balance" auth="bearer" />

---

## Resposta

<ResponseField name="balance" type="number">
  Saldo atual de créditos disponíveis para emissão de certificados.
</ResponseField>

<ResponseField name="profile_id" type="string">
  Identificador único do perfil do emissor.
</ResponseField>

<ResponseField name="company_name" type="string">
  Nome da empresa emissora.
</ResponseField>

---

## Exemplo de resposta

<ResponseExample>

```json Response (200)
{
  "balance": 150.5,
  "profile_id": "profile_123",
  "company_name": "Nome da Empresa"
}
```

</ResponseExample>

<Info>
  Caso a empresa não possua créditos disponíveis, o valor retornado em balance será 0.
</Info>
