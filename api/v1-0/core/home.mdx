A **API Pública v1.0** permite integrar sistemas externos à plataforma **That's Me** para criação de eventos, emissão de certificados e consulta de dados operacionais.

Esta API é destinada a:

- Plataformas de eventos
- Sistemas educacionais
- ERPs, CRMs e automações
- Backends próprios que desejam emitir ou consultar conquistas digitais

<Info>
  Todas as rotas exigem autenticação via API Key utilizando o header:

  <code>
  Authorization: Bearer sua_api_key
  </code>

  As API Keys são geradas nas configurações de contas emissoras dentro da plataforma That's Me.
</Info>
