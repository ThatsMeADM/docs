Retorna o saldo de créditos da empresa autenticada.

## Endpoint

<Request
  method="GET"
  url="/api/v1.0/{userCode}/balance"
  auth="bearer"
/>

## Response Fields

<ResponseField name="balance" type="number">
  Saldo atual de créditos da empresa em reais (R$).
</ResponseField>

<ResponseField name="profile_id" type="string">
  ID único do perfil da empresa.
</ResponseField>

<ResponseField name="company_name" type="string">
  Nome da empresa.
</ResponseField>

## Response Example

<Response status="200" type="object">
```json
{
  "balance": 150.5,
  "profile_id": "profile_123",
  "company_name": "Nome da Empresa"
}
</Response>
