## O que é o Saldo?

O **Saldo** representa os créditos disponíveis na conta do emissor para a **emissão de certificados** na plataforma ThatsMe.

Cada emissão de certificado consome créditos do saldo conforme as regras vigentes.

---

## Como o Saldo funciona

* O saldo é **gerenciado exclusivamente pela plataforma ThatsMe**
* Não é possível adicionar créditos via API
* O saldo é **consumido somente no momento da emissão do certificado**

---

## Consulta do Saldo

A API permite que o emissor:

* Consulte o saldo atual disponível
* Verifique se há créditos suficientes antes de realizar emissões

---

## Importante

* Cada emissor possui seu próprio saldo
* O saldo é atualizado em tempo real após cada emissão
* Emissões não são realizadas quando o saldo é insuficiente

<Warning>
Tentativas de emissão sem saldo disponível resultam em erro e não consomem créditos.
</Warning>
