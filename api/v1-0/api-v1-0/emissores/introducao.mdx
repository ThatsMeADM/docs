## O que são Emissores?

Os **Emissores** são empresas, organizações ou instituições que têm permissão para emitir certificados e badges na plataforma ThatsMe.

## Características dos Emissores

Cada emissor possui:
- **Perfil único**: Identificação única na plataforma
- **Informações da empresa**: Nome, razão social, setor de atuação
- **Credenciais de acesso**: API Keys para integração
- **Saldo de créditos**: Para aquisição de Badges (TM's) na Loja

## Funcionalidades dos Emissores

Os emissores podem:
- **Emitir certificados** para destinatários
- **Criar e gerenciar eventos**
- **Adquirir Badges** na Loja de TM's
- **Gerenciar destinatários** e convites
- **Visualizar estatísticas** de emissões
- **Integrar via API** para automação de processos

## Identificação

Cada emissor possui um **ID único** que é usado em:
- Endpoints da API para filtrar dados por emissor
- Associação de certificados e badges
- Relatórios e análises

<Note>
O userCode gerado ao criar a primeira API Key está associado ao perfil do emissor.
</Note>
