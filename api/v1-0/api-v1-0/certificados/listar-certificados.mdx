---
api: "GET /api/v1.0/assertions"
title: "Listar Certificados"
---

Retorna certificados emitidos pelo **emissor autenticado**, permitindo acompanhar o status de aceite e localizar emissões sem expor dados sensíveis de destinatários.

<Request method="GET" url="/api/v1.0/assertions" auth="bearer" />

<Note>
  Este endpoint não retorna dados pessoais do destinatário (ex.: nome, e-mail, nickname).
</Note>

---

## Parâmetros de consulta

<ParamField query="limit" type="number">
  Quantidade máxima de certificados retornados por página.
</ParamField>

<ParamField query="offset" type="number">
  Posição inicial da listagem para paginação.
</ParamField>

<ParamField query="status" type="string">
  Filtra pelo status de aceite do certificado. Valores aceitos: `PENDING`, `ACCEPTED`.
</ParamField>

<ParamField query="event_id" type="string">
  Filtra certificados emitidos para um evento específico.
</ParamField>

<ParamField query="badge_id" type="string">
  Filtra certificados emitidos utilizando um badge específico.
</ParamField>

<ParamField query="created_from" type="string">
  Filtra certificados emitidos a partir de uma data (ISO 8601).
</ParamField>

<ParamField query="created_to" type="string">
  Filtra certificados emitidos até uma data (ISO 8601).
</ParamField>

<ParamField query="sort" type="string">
  Campo de ordenação. Padrão recomendado: `created_at`.
</ParamField>

<ParamField query="order" type="string">
  Ordem de ordenação: `asc` ou `desc`. Padrão recomendado: `desc`.
</ParamField>

---

## Resposta

<ResponseField name="data" type="array">
  Lista de certificados emitidos pelo emissor autenticado.

  <Expandable title="items">
    <ResponseField name="assertion_id" type="string">
      Identificador único do certificado
    </ResponseField>
    <ResponseField name="status" type="string">
      Status de aceite (`PENDING` ou `ACCEPTED`)
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Data e hora da emissão (ISO 8601)
    </ResponseField>
    <ResponseField name="event_id" type="string">
      ID do evento associado à emissão
    </ResponseField>
    <ResponseField name="badge_id" type="string">
      ID do badge utilizado na emissão
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="number">
  Quantidade total de certificados encontrados
</ResponseField>

<ResponseField name="limit" type="number">
  Limite aplicado na consulta
</ResponseField>

<ResponseField name="offset" type="number">
  Offset aplicado na consulta
</ResponseField>

---
