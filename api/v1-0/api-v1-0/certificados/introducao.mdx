## Visão geral

**Certificados** são registros digitais emitidos para destinatários, representando conquistas, participações, reconhecimentos ou outras realizações.

Na That’s Me, a emissão cria um **convite**. O certificado só se torna **válido e ativo** após o destinatário **aceitar** o recebimento.

---

## Fluxo de emissão

### 1) Emissão (cria convite)

Ao emitir um certificado via API ou plataforma:

1. O sistema cria o registro do certificado (em estado de convite)
2. Um **convite é enviado automaticamente** ao destinatário
3. O destinatário deve acessar a plataforma e **aceitar** o recebimento

### 2) Ativação

Após a confirmação do destinatário:

* O certificado torna-se **oficial e válido**
* Pode ser verificado via mecanismos de validação
* Passa a aparecer no perfil do destinatário
* Pode ser compartilhado e visualizado publicamente

<Warning>
Os perfis possuem controle total sobre a privacidade de cada conquista, podendo limitar ou impedir a visualização pública dos certificados.
</Warning>
