## O que são Destinatários?

Os **Destinatários** são os usuários que recebem certificados e badges na plataforma ThatsMe. Na ThatsMe, os destinatários são os mesmos que os **ganhadores de um evento**.

## Relação com Eventos

Quando um evento é realizado:
- Participantes podem se tornar **ganhadores**
- Ganhadores são automaticamente registrados como **destinatários**
- Destinatários recebem convites para certificados relacionados ao evento

## Fluxo de Destinatários

### 1. Registro como Ganhador

Em um evento:
- Participantes são identificados como ganhadores
- Ganhadores são registrados no sistema como destinatários
- Informações do destinatário são associadas ao evento

### 2. Recebimento de Convites

Após a emissão de um certificado:
- O destinatário recebe um **convite** para aceitar o certificado
- O convite contém informações sobre o certificado e o evento
- O destinatário pode visualizar o convite pendente

### 3. Aceite e Ativação

Quando o destinatário aceita o convite:
- O certificado se torna **oficial e válido**
- Aparece no perfil do destinatário
- Pode ser compartilhado e verificado

## Informações dos Destinatários

Cada destinatário possui:
- **Perfil**: Nome, email, nickname
- **Histórico**: Certificados recebidos e aceitos
- **Convites pendentes**: Certificados aguardando aceite
- **Associação com eventos**: Eventos onde foi ganhador

<Note>
Os destinatários podem ser consultados por emissor, permitindo que organizações vejam todos os ganhadores de seus eventos.
</Note>
