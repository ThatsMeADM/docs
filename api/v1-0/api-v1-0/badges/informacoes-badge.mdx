---
api: "GET /api/v1.0/badges/{id}"
title: "Informações de uma Badge"
---

Retorna os detalhes completos de uma Badge (TM) específica do **emissor autenticado**.

<Request method="GET" url="/api/v1.0/badges/{id}" auth="bearer" />

<Note>
  Uso recomendado: após localizar o `badge_id` em **Listar Badges**, utilize este endpoint para carregar informações completas.
</Note>

---

## Parâmetros de rota

<ParamField path="id" type="string" required>
  ID do badge (`badge_id`).
</ParamField>

---

## Resposta

<ResponseField name="badge_id" type="string">
  ID do badge
</ResponseField>

<ResponseField name="name" type="string">
  Nome do badge
</ResponseField>

<ResponseField name="description" type="string">
  Descrição do badge
</ResponseField>

<ResponseField name="created_at" type="string">
  Data e hora de criação (ISO 8601)
</ResponseField>

<ResponseField name="assets" type="object">
  Assets visuais do badge

  <Expandable title="properties">
    <ResponseField name="image_url" type="string">
      URL da imagem do badge
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="user" type="object">
  Informações do criador

  <Expandable title="properties">
    <ResponseField name="profile" type="object">
      Perfil do criador

      <Expandable title="properties">
        <ResponseField name="profile_id" type="string">
          ID do perfil
        </ResponseField>
        ```
        <ResponseField name="company_name" type="string">
        Nome da empresa
        </ResponseField>
        ```
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

---
