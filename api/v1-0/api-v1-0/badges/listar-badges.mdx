---
api: "GET /api/v1.0/badges"
title: "Listar Badges"
---

Retorna uma lista **enxuta** de Badges (TM’s) do **emissor autenticado**, ideal para localizar rapidamente o `badge_id` para emissão.

<Request method="GET" url="/api/v1.0/badges" auth="bearer" />

---

## Resposta

<ResponseField name="data" type="array">
  Lista de badges do emissor autenticado.

  <Expandable title="items">
    <ResponseField name="badge_id" type="string">
      Identificador único do badge
    </ResponseField>
    <ResponseField name="name" type="string">
      Nome do badge
    </ResponseField>
    <ResponseField name="description" type="string">
      Descrição do badge
    </ResponseField>
    <ResponseField name="assets" type="object">
      Assets visuais do badge

      <Expandable title="properties">
        <ResponseField name="image_url" type="string">
          URL da imagem do badge
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="number">
  Quantidade total de badges encontrados
</ResponseField>

<ResponseField name="limit" type="number">
  Limite aplicado na consulta
</ResponseField>

<ResponseField name="offset" type="number">
  Offset aplicado na consulta
</ResponseField>

---
