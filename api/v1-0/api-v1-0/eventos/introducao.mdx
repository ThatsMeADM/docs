## Visão geral

**Eventos** são o recurso central da API Pública da That’s Me.
Eles representam qualquer contexto no qual **certificados e badges (TM's)** podem ser emitidos, como competições, cursos, treinamentos, workshops ou premiações.

Todo certificado emitido pela API está **sempre associado a um evento**.

---

## O que um Evento representa

Cada evento reúne as informações necessárias para organizar e emitir conquistas digitais, incluindo:

* **Identificação**: nome, descrição e período
* **Localização**: online ou presencial (cidade e estado)
* **Classificação**: classificação do tipo de evento
* **Badges (TM's)**: Insignias que poderão ser emitidas
* **Status**: estado atual do evento (`Active` ou `Concluído`)

Esses dados são utilizados tanto para emissão de certificados quanto para indexação e recomendação dentro da plataforma.

---

## Tipos de Eventos

Os eventos podem pertencer a diferentes categorias, de acordo com seu contexto:

* **Esporte**
  Competições esportivas, corridas, torneios e e-sports

* **Educação**
  Cursos, treinamentos, workshops e formações

* **Corporativo**
  Eventos empresariais e reconhecimento profissional

* **Sociocultural**
  Eventos culturais, sociais e comunitários

---

## Operações disponíveis na API

Através da API, é possível:

* Criar novos eventos
* Atualizar eventos existentes
* Consultar detalhes operacionais
* Emitir certificados

Cada operação está documentada nos endpoints desta seção.

---

## Eventos e certificados

Os eventos funcionam como a **base para emissão de certificados**:

* Badges (TM's) são vinculados a eventos
* Certificados são emitidos a partir dessas associações
* Destinatários (ganhadores) sempre pertencem a um evento específico

<Note>
Eventos podem ser **públicos** (visíveis na plataforma) ou **privados** (visíveis apenas para o emissor).
</Note>
