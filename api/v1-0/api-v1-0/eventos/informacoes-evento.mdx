---
api: "GET /api/v1.0/events/{id}"
title: "Informações de um Evento"
---

Retorna os **detalhes completos** de um evento específico do **próprio emissor autenticado**.

Este endpoint exige autenticação via **Bearer Token** e não permite acessar eventos de outros emissores.

<Request method="GET" url="/api/v1.0/events/{id}" auth="bearer" />

<Note>
  Uso recomendado: após localizar o `event_id` em **Listar Eventos**, utilize este endpoint para carregar os detalhes completos do evento.
</Note>

---

## Parâmetros de rota

<ParamField path="id" type="string" required>
  ID do evento (`event_id`).
</ParamField>

---

## Resposta

<ResponseField name="event_id" type="string">
  ID do evento
</ResponseField>

<ResponseField name="name" type="string">
  Nome do evento
</ResponseField>

<ResponseField name="description" type="string">
  Descrição do evento
</ResponseField>

<ResponseField name="category" type="string">
  Categoria do evento
</ResponseField>

<ResponseField name="subcategory" type="string">
  Subcategoria do evento
</ResponseField>

<ResponseField name="city" type="string">
  Cidade do evento
</ResponseField>

<ResponseField name="state" type="string">
  Estado (UF)
</ResponseField>

<ResponseField name="initial_date" type="string">
  Data e hora de início (ISO 8601)
</ResponseField>

<ResponseField name="finish_date" type="string">
  Data e hora de término (ISO 8601)
</ResponseField>

<ResponseField name="image_url" type="string">
  URL da imagem do evento
</ResponseField>

<ResponseField name="official_url" type="string">
  URL oficial do evento
</ResponseField>

<ResponseField name="status" type="string">
  Status do evento (ex.: ACTIVE or COMPLETED)
</ResponseField>

<ResponseField name="profile" type="object">
  Informações do organizador

  <Expandable title="properties">
    <ResponseField name="name" type="string">
      Nome do organizador
    </ResponseField>
    <ResponseField name="company_name" type="string">
      Nome da empresa organizadora
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="badges" type="array">
  Badges associados ao evento

  <Expandable title="items">
    <ResponseField name="badge_id" type="string">
      ID do badge
    </ResponseField>
    <ResponseField name="name" type="string">
      Nome do badge
    </ResponseField>
    <ResponseField name="description" type="string">
      Descrição do badge
    </ResponseField>
    <ResponseField name="assets" type="object">
      Assets do badge

      <Expandable title="properties">
        <ResponseField name="image_url" type="string">
          URL da imagem do badge
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
  <Note>
    Os badges associados ao evento são retornados no campo `badges` desta resposta.\
    Não é necessário chamar um endpoint separado para obter essa informação.
  </Note>
</ResponseField>

---
