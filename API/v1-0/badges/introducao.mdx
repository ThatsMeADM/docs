## O que são Badges?

Os **Badges** na plataforma ThatsMe são os **TM's** (That's Me), que são modelos de certificados e conquistas que podem ser emitidos para destinatários.

## Como funcionam os Badges

Os Badges são **modelos pré-configurados** de certificados que contêm:
- Design personalizado
- Informações sobre a conquista ou certificação
- Metadados e configurações específicas

## Aquisição de Badges

Os Badges (TM's) são **adquiridos pela Loja de TM's** dentro da plataforma ThatsMe. Os emissores podem:

1. Navegar pela Loja de TM's
2. Selecionar e adquirir os modelos desejados
3. Usar os Badges adquiridos para emitir certificados aos destinatários

## Uso dos Badges

Após adquirir um Badge na Loja de TM's, ele fica disponível para:
- Emissão de certificados em eventos
- Criação de assertions (certificados emitidos)
- Personalização e configuração conforme necessário

<Note>
Cada Badge adquirido pode ser usado múltiplas vezes para emitir certificados para diferentes destinatários.
</Note>
