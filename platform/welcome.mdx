That's Me is a platform that lets organizations issue verified digital certificates as 3D badges.
Recipients collect them on a public profile. The platform's recommendation engine
suggests the next relevant course, competition, or certification — driving engagement and upsell.

## Core concepts

| Concept | Description |
|---|---|
| **Event** | A course, competition, or activity that generates certificates |
| **Badge** | The 3D visual template of the certificate |
| **Certificate** | An issued credential linking a recipient to an event |
| **Profile** | A recipient's public page with all their achievements |
| **Trail** | A sequence of connected events (e.g. English Level 1 → 2 → 3) |
| **Verified issuer** | An organization verified via KYB with a checkmark on all certificates |

## Anti-fraud

Every certificate issued on That's Me receives a SHA-256 cryptographic hash.
Anyone can verify authenticity at `thatsme.com.br/certificate/verify/[hash]`.
Certificates from verified issuers include the KYB verification ID in the hash —
immutable proof that the issuer was verified at emission time.
