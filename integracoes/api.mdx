A API Pública v1.0 permite integração com o sistema ThatsMe usando API Keys. Todas as rotas requerem autenticação via Bearer Token.

<Info>
Todas as rotas desta API requerem o parâmetro `userCode` na URL, que é o código único da empresa gerado automaticamente ao criar a primeira API Key.
</Info>

## Autenticação

Todas as requisições devem incluir o header de autenticação:

```bash
Authorization: Bearer sua_api_key_aqui
```

---

## GET /api/v1.0/:userCode

Retorna informações sobre a API v1.0.

**Resposta 200:**

```json
{
  "message": "v1.0 API",
  "version": "1.0"
}
```

---

## GET /api/v1.0/:userCode/balance

Retorna o saldo de créditos da empresa autenticada.

<RequestExample>

```bash
curl -X GET "https://api.thatsme.com.br/api/v1.0/{userCode}/balance" \
  -H "Authorization: Bearer sua_api_key"
```

</RequestExample>

**Resposta 200:**

```json
{
  "balance": 150.50,
  "profile_id": "profile_123",
  "company_name": "Nome da Empresa"
}
```

<Info>
O saldo retornado é o saldo atual de créditos da empresa. Se a empresa não tiver registro de créditos, o saldo será 0.
</Info>

## POST /api/v1.0/:userCode/assertions

Emite uma assertion (certificado).

<ParamField path="body" type="object">
  <ParamField path="badge_id" type="string" required>
    ID do badge
  </ParamField>
  <ParamField path="event_id" type="string" required>
    ID do evento
  </ParamField>
  <ParamField path="recipient_email" type="string" required>
    Email do recebedor
  </ParamField>
  <ParamField path="recipient_name" type="string" required>
    Nome do recebedor
  </ParamField>
  <ParamField path="fields" type="object">
    Campos personalizados do certificado
  </ParamField>
</ParamField>

<RequestExample>

```bash
curl -X POST "https://api.thatsme.com.br/api/v1.0/{userCode}/assertions" \
  -H "Authorization: Bearer sua_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "badge_id": "badge_123",
    "event_id": "event_456",
    "recipient_email": "usuario@exemplo.com",
    "recipient_name": "João Silva"
  }'
```

</RequestExample>

**Resposta 201:**

```json
{
  "post_id": "post_123",
  "message": "Assertion emitida com sucesso",
  "recipient": {
    "email": "usuario@exemplo.com",
    "name": "João Silva"
  },
  "badge": {
    "badge_id": "badge_456",
    "name": "Certificado de Participação"
  },
  "event": {
    "event_id": "event_789",
    "name": "Workshop de Tecnologia"
  }
}
```
