The Badge Gallery is where you manage all 3D digital trophies on your issuer account. Here you can view, organize, edit, and prepare TMs for use in issuances.

## Features

### Viewing and organization

- **TM grid** — all published models on the account, with search and sorting
- **Search** — find TMs by name
- **Sort** — newest, oldest, name A–Z or Z–A
- **Fullscreen** — immersive 3D trophy viewing in the browser
- **Copy ID** — copy the TM identifier for API usage

### Metadata editing

Each TM's **name** and **description** can be edited directly in the gallery. This is essential for teams with multiple administrators — making it easy to identify and correctly use TMs across different events.

<Tip>
Add clear descriptions and standardized names to your TMs to streamline your team's work during events with multiple simultaneous issuances.
</Tip>

### Deletion

TMs can only be deleted if they have **no linked issuances**. After the first issuance, the TM is permanently associated with the generated certificates.

## TMs pending approval

New models — created in the editor, by the design team, or imported — appear in a separate section at the top of the gallery. For each pending TM, two actions are available:

| Action | Description |
|--------|-------------|
| **Accept** | Officially adds the TM to the gallery for use in events |
| **Report technical issue** | Reports visual errors or defects in the model |

<Warning>
Only TMs with **Accepted** status are available for issuance. The revision request option is available only for **Exclusive** TMs.
</Warning>

<Info>
All TMs are linked to the issuer account and can be used by any authorized administrator on the team.
</Info>
