## Native sharing options

That's Me offers **native, smart sharing of digital certificates**, designed for reach and credibility. The feature is fast, intuitive, and compatible with all major digital channels — everything from a single click.

## How it works

When viewing any certificate or digital trophy, click **"Share Certificate"**. A sharing panel appears with:

- One-click icons for **Facebook**, **LinkedIn**, **X (Twitter)**, **WhatsApp**, **Email**, and **Pinterest**
- A **unique, verifiable, public URL** for the achievement
- A quick-copy button for the link

<Info>
Each sharing channel opens with pre-filled content optimized for that platform. No manual formatting needed.
</Info>

## Link structure and metadata

Each certificate has a unique public link:

```
https://thatsme.com.br/certificate/{ID}
```

This link is public, traceable, and requires **no login** to view — making external engagement seamless.

When shared on social media, the link automatically generates a **rich preview** with the following metadata:

| Metadata | Content |
|----------|---------|
| **og:title** | Check out [User Name]'s Achievement |
| **og:description** | Recognition issued by [Organization Name]. View in 3D |
| **og:image** | Trophy cover image (rendered thumbnail) |
| **og:url** | Direct certificate link |
| **og:type** | website |

## What the shared post looks like

When someone clicks a shared link, they see:

- The **TM cover image** as visual preview
- **Recipient name**
- **Event or achievement name**
- Direct link to view the full interactive certificate

<Tip>
This structure is ideal for promoting recognition professionally — whether for sports events, educational programs, cultural activities, or corporate achievements.
</Tip>

## Copy link

Beyond native sharing buttons, you can copy the link and paste it anywhere: newsletters, emails, landing pages, websites, or any other channel.

<Note>
The link follows a universal pattern for public viewing and validation. It can be accessed by anyone, at any time, without authentication.
</Note>
