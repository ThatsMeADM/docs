Every achievement on That's Me automatically generates a **unique, permanent page** at `app.thatsme.com.br/c/[id]`. This link can be shared on resumes, social media, or any channel where you want to present an achievement with context and authenticity.

## Page structure

### Main section

The top section of the page displays the 3D trophy as a visual highlight, along with the key achievement information:

- **Recipient name** — with a link to the profile
- **Event name** — the context of the achievement
- **Date, location, and category** — issuance metadata
- **Celebrate button** — any visitor can send a confetti animation and register a like
- **Share button** — direct access to sharing channels

### Issuer card

Panel with the issuer's avatar, name, and bio. If the issuer has KYB verification, a **verification badge** appears next to their name, indicating that the organization was verified at the time of issuance.

### Reason for recognition

Text written by the issuer explaining the reason for the achievement.

### Certificate details

Grid with the certificate's technical information:

| Field | Content |
|-------|---------|
| Recipient | Name with a link to the profile |
| Achievement | Event name |
| Date | Issuance date |
| Location | City and state |
| Category | Event category |
| Digital TM | Button to open the interactive 3D trophy in fullscreen |
| Validity | Permanent or with an expiration date |

### Authenticity

Dedicated verification section with:
- **SHA-256 hash** — unique, immutable code generated from the certificate data
- **QR code** — links directly to the verification page
- **Copy button** — to copy the hash

<Info>
Anyone can verify a certificate's authenticity via the hash or QR code, without needing to log in.
</Info>

### Next step

If the event has connections to other events or learning trails, the page may display a recommended next step with a direct link.

### About the event

Detailed event information, including recommendations for similar events based on the platform's recommendation engine.

### Other recipients

Grid with avatars of other participants recognized in the same event.

## Permanence

Certificates issued on That's Me are permanent. Once issued, they remain active and verifiable, even if:

- The issuer's plan is downgraded or cancelled
- The event is no longer active
- The issuer loses KYB verification

The SHA-256 hash is immutable. Certificates issued during a period when the issuer was verified retain that record associated with the issuance.

## Use as portfolio

<Tip>
Use the achievement link on resumes, proposals, LinkedIn, online portfolios, or anywhere you need to prove a skill, participation, or experience with greater credibility.
</Tip>
