Every achievement on That's Me automatically generates a **unique, permanent page** at `thatsme.com.br/c/[id]`. This link can be shared on resumes, social media, or anywhere you want to display an achievement with context and authenticity.

## Page structure

### Hero section

The top section displays the 3D trophy as a visual backdrop, with the core achievement information:

- **Recipient name** — with link to their profile
- **Event name** — the context of the achievement
- **Date, location, and category** — issuance metadata
- **Celebrate button** — any visitor can send a confetti animation and register a like
- **Share button** — direct access to all sharing channels

### Issuer card

Panel with the issuer's avatar, name, and bio. If the issuer has KYB verification, a **verification badge** appears next to their name — proof the organization was verified at the time of issuance.

### Reason for recognition

Text written by the issuer explaining why the achievement was granted. The section is expandable for longer texts.

### Certificate details

Grid with technical information:

| Field | Content |
|-------|---------|
| Recipient | Name with profile link |
| Achievement | Event name |
| Date | Issuance date |
| Location | City and state |
| Category | Event category |
| Digital TM | Button to open the interactive 3D trophy in fullscreen |
| Validity | Permanent or with expiration date |

### Authenticity

Dedicated verification section with:
- **SHA-256 hash** — unique, immutable code generated from the certificate data
- **QR code** — links directly to the verification page
- **Copy button** — to copy the hash

<Info>
Anyone can verify a certificate's authenticity via the hash or QR code, without needing to log in.
</Info>

### Next step

If the event has **connections to other events** (learning trails), the page displays the recommended next step with a direct link.

### Event details

Detailed event information, including AI-powered recommendations for similar events.

### Other recipients

Avatar grid of other participants awarded in the same event.

## Permanence

Certificates on That's Me are **permanent**. Once issued, they remain active and verifiable forever — even if:

- The plan is downgraded or cancelled
- The event is deleted
- The issuer loses their KYB verification

| What | Status after cancellation |
|------|--------------------------|
| Existing certificates | Active and verifiable |
| Certificate pages | Accessible via direct link |
| SHA-256 verification | Works permanently |
| Recipient profiles | Certificates remain visible |
| New issuances | Blocked (requires active plan) |
| Public directory listing | Active for 90-day grace period |

The SHA-256 hash is immutable. Certificates issued during a KYB verification period retain the verification ID in the hash — permanent proof the issuer was verified at the time of issuance.

### Why this matters

| Traditional approach | That's Me |
|---------------------|-----------|
| PDF can be forged | SHA-256 hash prevents tampering |
| Certificate breaks if platform shuts down | Hashes are stored independently |
| No way to verify issuer legitimacy | KYB verification embedded in the hash |
| Certificate exists in isolation | Connected to profile and recommendation engine |
