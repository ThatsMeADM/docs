## O que é MCP?

MCP (*Model Context Protocol*) é um protocolo aberto que permite que assistentes de IA, como Claude, ChatGPT e Cursor, acessem ferramentas externas de forma padronizada.

A That's Me oferece dois servidores MCP:

| Servidor | Público | Função |
|---|---|---|
| **@thatsme/mcp** | Sim | Buscar eventos e recomendar próximos passos |
| **thatsme-mcp-issuer** | Requer API Key | Emitir certificados, consultar analytics e exportar dados |

## MCP Público — @thatsme/mcp

O MCP público permite que qualquer assistente de IA busque eventos da plataforma. Não requer autenticação.

### Instalação

```bash
npx -y @thatsme/mcp
```

### Tools disponíveis

| Tool | Descrição |
|---|---|
| `search_thatsme_events` | Busca eventos por nome, categoria, nível e localização |
| `get_event_next_steps` | Dado um evento ou skill, sugere próximos passos |

### Exemplo de prompt

> "Encontre cursos de Python para iniciantes no That's Me"

O assistente usa `search_thatsme_events` para buscar eventos com `query: "Python"`, `level: "beginner"` e `category: "EDUCATION"`.

## MCP Emissor — thatsme-mcp-issuer

Voltado para emissores que desejam gerenciar certificados por IA. Requer API Key.

[Documentação completa do MCP Emissor →](/mcp/mcp-emissor)

## Compatibilidade

| Assistente | MCP Público | MCP Emissor |
|---|---|---|
| Claude Desktop | stdio | stdio |
| Claude Code | stdio | stdio |
| Cursor | stdio | stdio |
| Remoto (qualquer) | — | HTTP/SSE via `https://mcp.thatsme.com.br/mcp` |
