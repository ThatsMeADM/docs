A API Pública v1.0 permite integração com o sistema ThatsMe usando **API Keys**.

<Info>
  Todas as rotas requerem autenticação via <code>
  Authorization: Bearer sua_api_key</code>

   e exigem <code>
  userCode</code>

   no path.
</Info>

## Comece por aqui

- **Autenticação**: envie o header `Authorization: Bearer sua_api_key`
- **Base URL**: `https://api.thatsme.com.br`
- **Path obrigatório**: `{userCode}` (código único da empresa gerado ao criar a primeira API Key)
